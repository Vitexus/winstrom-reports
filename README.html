Jak začit s reporty

Základem reportů je nástroj iReport. Je potřeba jeho stará verze 2.0.2.

  1. Stáhněte iReport z adresy: http://download.flexibee.eu/download/report/iReport-2.0.2.zip
  2. Rozbalte jej
  3. Nakopírujte soubor config do:
    • Linux: $HOME/.ireport/
    • Windows: C:\Documents and Settings\${user}\.ireport
  4. Stáhněte si zdrojáky reportů z adresy http://download.flexibee.eu/download/latest/reports/
  5. Otevřete požadovaný report (např. faktura vydaná - favAkt/fakturyKB.jrxml)
  6. Upravte jej a uložte
  7. Spusťte FlexiBee, přihlašte se do firmy a spusťte Nástroje / Uživatelské reporty. Pokud položka není vidět, požádejte podporu o bezplatnou úpravu licence
  8. Založte nový uživatelský report:
    • Zkratka - unikátní strojový název nového reportu (např. "upravenaFaktura")
    • Název - text zobrazený uživateli
    • Kód formuláře - odpovídá adresáři, ze kterého byl report vzat (např. favAkt)
  9. Předvybraný počet záznamů nastavte na Aktuální záznam.
  10. Do příloh nahrajte změněný report a také vše z podadresáře subreports
  11. Uložte vše a znovu otevřete report.
  12. Nastavte hlavní report na faktura.jasper.

Po uložení už bude daný report přístupný v dané sekci. Pokud chcete změnit tento report na výchozí, lze to upravit u typu dokladu.

Pokud chcete přenést report ke klientovy, použijte XML exporty a importy.
